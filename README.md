# llm-study (공부 노트)

AI/LLM을 처음부터 차근차근 이해하기 위한 “개인 공부 노트”입니다. 프로젝트가 아니라 개념과 용어를 익히고, 작게 실습하며, 배운 것을 짧게 기록하는 것을 목표로 합니다.

## 이 저장소의 목적
- **LLM이 뭔지부터** 차근히 이해하기
- **어려운 수식/코드 없이** 먼저 개념부터 잡기
- **필요할 때만** 가볍게 실습(노트 수준)
- 배운 내용을 **짧게 요약**해서 나만의 언어로 정리

## LLM을 아주 쉽게 말하면
- **LLM**: 문장을 보고 다음에 올 글자를(토큰) 가장 그럴듯하게 예측하는 모델
- **왜 중요한가**: 질문에 답하기, 번역하기, 요약하기 등 “문자 관련 작업”을 폭넓게 잘함
- **중요한 요소 몇 가지**
  - 토큰: 글자를 모델이 다루기 쉬운 단위로 쪼갠 것
  - 임베딩: 단어/토큰을 숫자 벡터로 바꾼 표현
  - 어텐션: 입력 중 “무엇이 중요한지” 가중치를 두고 보는 방식

## 처음에 볼 만한 질문들
- “토큰”은 왜 필요한가?
- “임베딩”은 무슨 의미인가?
- “어텐션”은 무엇을 더 잘하게 해주는가?
- “프롬프트”는 왜 결과에 큰 영향을 줄까?

## 공부 방식(가볍게)
1) 개념 3줄 요약 → 2) 예시 한 줄 → 3) 내가 이해한 말로 다시 쓰기  
필요하면 간단한 그림/메모만. 환경 설치나 복잡한 디렉토리는 “나중에”.

## 아주 작은 실습 아이디어(선택)
- 공개 챗봇/모델에 같은 질문을 다르게 물어보기(프롬프트 효과 보기)
- 글 1문단 → 1문장 요약해보기(사람 방식 vs 모델 결과 비교)

## 용어 미니 사전(계속 추가)
- **토큰(Token)**: 문장을 잘게 나눈 단위(단어/부분 단어 등)
- **임베딩(Embedding)**: 토큰을 고정 길이 숫자 벡터로 표현한 것
- **어텐션(Attention)**: 입력의 어떤 부분이 중요한지 가중치를 두는 메커니즘
- **프롬프트(Prompt)**: 모델에게 건네는 입력 지시/질문

---

이 저장소는 “공부 노트”입니다. 부담 없이 짧게, 자주 기록하며 개념부터 천천히 익힙니다. 필요해지면 그때 가볍게 실습을 덧붙입니다.
